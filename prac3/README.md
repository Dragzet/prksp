# Практическая работа №3 - Пропсы и маршруты в React

## Описание проекта

Интернет-магазин электроники, созданный с использованием React и React Router.

## Реализованные компоненты

### Компоненты с props:
- **ProductCard** - карточка товара (props: id, name, price, image, description)
- **ProductList** - список товаров
- **ProductDetail** - детальная страница товара с использованием useParams
- **Navbar** - навигационное меню

### Страницы:
- **Home** - главная страница с каталогом товаров
- **About** - страница "О нас"
- **Contacts** - страница контактов
- **NotFound** - страница 404

## Маршрутизация

Реализованы следующие маршруты:
- `/` - главная страница
- `/about` - страница "О нас"
- `/contacts` - контакты
- `/product/:id` - детальная страница товара (динамический параметр)
- `*` - страница 404 для несуществующих маршрутов

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите приложение:
```bash
npm start
```

Приложение откроется в браузере по адресу [http://localhost:3000](http://localhost:3000)

## Используемые технологии

- React 18
- React Router DOM v6
- CSS3

## Особенности реализации

- Использование компонента `Link` вместо `<a>` для навигации без перезагрузки страницы
- Передача данных через props
- Динамические маршруты с параметрами (useParams)
- Программная навигация с useNavigate
- Обработка несуществующих страниц (404)
